Welcome to Drawing App 2016!

Draw whatever you like! Below you will find a set of 21 colors, three set of control buttons (Thickness, Opacity and Linecap) and two special buttons (Clear and Save).

Thickness changes the line's stroke width.

Opacity changes the line's opacity.

Linecap changes the line's shape of the ends.

Clear clears the canvas.

Save exports your current canvas drawing to a PNG image (canvas.png).

Enjoy!
